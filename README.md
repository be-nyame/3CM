# 3CM - Crowd Counting for Crowd Management

## Given a photo of the crowd, this crowd counting app can estimate the number of people present. The app also keeps track of crowd counts of recorded events.

- ### Environment
  Windows 10

- ### Requirements
  - #### Web Stack
    react 17.0.2 <br>
    express 4.17.1 <br>
    mongoose 5.12.3 <br>
    tailwind <br>
    flask 
  - #### Machine Learning
    pytorch <br>
